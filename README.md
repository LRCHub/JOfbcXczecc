# Harumi – 記憶の海

> **歌詞登録ステータス：プレーン歌詞のみ**

```
ねぇ、覚えてる?
柔らかい光に抱かれて
二人過ごした日々
そう、このままで
いられると思ってた
疑いもせずに
君だけを見つめていた
満ちては返す波に 何を願っていたの
果てのないこの海を 一人で進み出すの
闇の中 漂う小舟のように
どこにもいない君を どうして求めてるの
手を伸ばして
記憶の海に君を探すの
ねぇ、聞こえてる?
この海は あの日も今日も
同じ歌を歌っていた
夢から覚めたはずの 今でも
君を覚えているよ
果てのないこの海を 一人で進み出すの
ただ 時の流れに揺られるままに
代わりのいない君を 心は求めてるの
手を伸ばして
記憶の海に君を探すの
果てのないこの海を 一人で進めたなら
この涙も 記憶の海に溶けて消えるの
```
